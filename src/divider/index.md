# Divider 分割线

我们提出的分割线可以设置6种颜色，支持在分割线中间添加文字。

## API

| 参数 | 说明 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| style | 分割线的颜色 | `color`\|`hover`| `hover` |
| items | 菜单配置项 | `itemProps` | - |
| children | 菜单内容 | `ReactNode` | - |
| onChange | 菜单显示时状态变化触发的函数 | `Function` | - |
