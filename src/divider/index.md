## API

| 参数 | 说明 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| style | 分割线的颜色 |`primary`\|`success`\|`info`\|`warning`\|`danger`\|`link`| `primary` |
| text | 分割线中间的文本 | `string` | - |
