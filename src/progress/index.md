## API

### Menu
| 参数 | 说明 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| percent | 百分比 | `number` | - |
| max | 最大值 | `number` | - |
| theme | 主题类型 | `primary`\|`success`\|`warning`\|`danger`\|`link` | `primary` |
| inner | 进度条内显示文字 | `boolean` | `false` |
| circle | 圆形进度条 | `boolean` | `false` |
