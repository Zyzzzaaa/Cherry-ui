## API

### Menu
| 参数 | 说明 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| mode | 菜单类型 | `horizontal`\|`vertical`| `horizontal` |
| defaultIndex | 默认选中的菜单项的索引值 | `string` | `0` |
| onSelect | 被选中时触发的回调函数 | `function` | - |
| className | 自定义类名 | `string` | - |
| style | 自定义样式 | `object` | - |
