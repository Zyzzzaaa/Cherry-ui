## API

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :---: | :---: | :---: | :---: | :---: |
| data | 树结构数据 | `Array<TreeNode<T>>` | - | - |
| data.value | 节点的真值 | `T` | - | - |
| data.label | 节点的显示值 | `string` | - | - |
