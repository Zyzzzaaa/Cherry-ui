## API

| 参数 | 说明 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| data | 树结构数据 | `Array<Node>` | - |
| defaultCheckedKeys | 默认选中项 | `number[] \| string[]` | `[]` |
| defaultExpand | 默认展开项 | `number[] \| string[]` | `[]` |
| defaultExpandAll | 是否默认打开全部选择菜单 | `boolean` | `false` |
| height | 选择菜单高度 | `string` | - | - |
