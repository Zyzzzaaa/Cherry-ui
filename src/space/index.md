# Space 间距

我们提出间距组件来避免子组件紧贴在一起，可以快速的设置组件排列样式，使组件之间拉开相同的距离。

## API

| 参数 | 说明 | 类型 | 默认值 |
| :-- | :-- | :-- | :-- |
| side | 子元素对齐方式 |`left`\|`center`\|`right`| `left` |
| direction | 子元素排列的方向 |`row`\|`column`| `row` |
| size | 子元素的间距 |`small`\|`medium`| `large` |
